# PRODIGY_DS_05

## Traffic Accident Data Analysis

### Objective
Analyze traffic accident data to identify patterns related to road conditions, weather conditions, and time of day. Visualize accident hotspots and contributing factors.

### Dataset
Accident Sample Dataset (5000 records)

### Tasks Performed
- Data Cleaning
- Accident Severity Analysis
- Weather Condition Analysis
- Road Surface Condition Analysis
- Time of Day Analysis
- Urban vs Rural Accident Comparison
- Accident Hotspot Visualization

### Visualizations
1. Accident Severity Distribution
2. Weather Conditions During Accidents
3. Road Surface Conditions
4. Accidents by Hour of Day
5. Urban vs Rural Accidents
6. Accident Hotspots

### Key Insights
- Most accidents were classified as Slight.
- Fine weather conditions recorded the highest number of accidents.
- Dry roads accounted for the majority of accidents.
- Accident frequency peaked during evening traffic hours.
- Urban areas experienced more accidents than rural areas.
- Hotspot visualization revealed accident-prone regions.

### Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
